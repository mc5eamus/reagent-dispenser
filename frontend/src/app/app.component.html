Welcome to the Reagent Dispenser Application

This application provides an interface for managing lab reagent dispensing operations with 96-well plate (8×12 format) management.

Features:

  • Manage laboratory plates with unique barcode identification
  • Track wells within plates using standard notation (A1-H12)
  • Manage reagent inventory and properties
  • Execute and monitor dispense operations in real-time
  • Track operation status lifecycle (PENDING → IN_PROGRESS → COMPLETED/FAILED)
